# iOS 捷徑設定教學

## 功能
每天自動從 Apple Health 讀取健康數據，傳送到你的 Fitness Coach 後端。

## 設定步驟

### 1. 建立新捷徑

打開 iPhone 的「捷徑」app → 點右上角 `+` → 命名為「同步健康數據」

### 2. 加入以下動作

依序加入：

1. **尋找健康樣本** → 類型：體重 → 排序：開始日期（最新優先）→ 限制：1
   - 設定變數名稱為 `weight`

2. **尋找健康樣本** → 類型：體脂肪百分比 → 排序：開始日期（最新優先）→ 限制：1
   - 設定變數名稱為 `bodyFat`

3. **尋找健康樣本** → 類型：步數 → 排序：開始日期（最新優先）→ 限制：1
   - 設定變數名稱為 `steps`

4. **尋找健康樣本** → 類型：靜止心率 → 排序：開始日期（最新優先）→ 限制：1
   - 設定變數名稱為 `heartRate`

5. **尋找健康樣本** → 類型：活動能量 → 排序：開始日期（最新優先）→ 限制：1
   - 設定變數名稱為 `activeCal`

6. **取得 URL 的內容**
   - URL: `https://你的render網址.onrender.com/api/health-data`
   - 方法: POST
   - 請求主體: JSON
   - 內容:
     ```json
     {
       "date": "目前日期（格式化為 yyyy-MM-dd）",
       "weight": weight 的數值,
       "body_fat_pct": bodyFat 的數值,
       "steps": steps 的數值,
       "resting_heart_rate": heartRate 的數值,
       "active_calories": activeCal 的數值
     }
     ```

### 3. 設定自動化

1. 打開「捷徑」app → 底部選「自動化」
2. 點 `+` → 建立個人自動化
3. 選「特定時間」→ 設定每天早上 8:00
4. 動作選「執行捷徑」→ 選擇「同步健康數據」
5. 關閉「執行前先詢問」

### 注意事項

- 第一次執行時，iOS 會要求你授權讀取健康數據，請全部允許
- PICOOC 的數據會自動同步到 Apple Health，所以這個捷徑也會抓到體脂計的數據
- 如果某項數據沒有（例如當天沒量體重），那個欄位會是 null，後端會自動忽略
